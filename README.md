# 🐎 Karabakh Horses — Visual Gallery & Info App

A fully responsive web application showcasing **Karabakh horses**, built with **React**.  
The app allows users to explore the beauty, history, and famous breeds of Karabakh horses, including an interactive gallery and embedded video content.

## 🚀 Live Demo
Check out the live project here:  
[View Live Project](https://karabakh-horses.vercel.app/) 🔗

---

## 📌 Overview

**Karabakh Horses** is a modern web app dedicated to celebrating the legendary Karabakh horse breed.  
The project focuses on building a polished front-end experience with responsive layouts, visually appealing galleries, and an engaging user interface.

Main features include:

- **Visual gallery** with lightbox view
- **Famous Karabakh horse profiles**
- **Embedded YouTube video** showing horses in action
- Smooth scroll animations and transitions

---

## ✨ Features

- Fully responsive design for **mobile, tablet, and desktop**
- **Interactive image gallery** with lightbox view
- Detailed profiles of **famous Karabakh horses**
- Embedded **YouTube video** showcasing horses in natural settings
- Smooth **scroll animations** and in-view transitions
- Reusable component-based architecture with **React hooks**
- State management using `useState` and `useRef`
